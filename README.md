 ## 功能
 * 消息自动回复
 * 消息防撤回
 * 自添加按钮
 * 聊天置顶
 *  开启忙碌模式
 * 解除大图片上传限制
 * 自动登录开关
 * 通知中心快捷回复
 * 插件版本提示
